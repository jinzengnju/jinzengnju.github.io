Ubuntu安装

Ubuntu安装

各分区使用考量，比如你安装软件多的话，就把系统目录给大点。但是如果你自己的数据文件比较多，那就把home目录给大点。

sda1:/boot 主分区 200M：用于存放启动Linux时使用的一些核心文件

sda2: / 逻辑分区 30G：用于存放系统，相当于Windows中的C盘，根目录

sda3: /swap 逻辑分区 内存两倍

sda6:/home 逻辑分区 剩余的空间：用户主目录，home目录下可能有多个工作用户。如果用户自身的数据较多，则推荐/home分区大一些

有些时候，在虚拟机上安装Ubuntu后，界面不能完全显示，遇到这种情况，可以安装VMtools即可。

在VMWARE光盘下：

mkdir /vmtools

tar xvf VMwareTools-10.0.0-2977863.tar.gz -C /vmtools

cd /vmtools/

ls

cd vmware-tools-distrib/

./vmware-install.pl
